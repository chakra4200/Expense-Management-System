# Expense Management System

## 📌 Project Description

The Expense Management System is a console-based application developed using Python and MySQL. It helps users manage their expenses by storing user information, expense categories, and expense records in a relational database.

This project demonstrates database connectivity, multiple table relationships, and SQL operations using Python.

---

## 🚀 Technologies Used

- Python
- MySQL
- MySQLdb
- SQL

---

## ✨ Features

- Add new users
- View all users
- Search users
- Add expense categories
- View expense categories
- Add expenses
- Search expenses using User and Category
- Relational database with Foreign Keys
- INNER JOIN queries

---

## 🗄️ Database Tables

1. Users
2. Category
3. Expenses

The **Expenses** table is linked with the **Users** and **Category** tables using Foreign Keys.

---

## ▶️ How to Run

1. Install Python.
2. Install MySQL.
3. Create a MySQL database.
4. Update the database connection details in `database.py`.
5. Run `database_setup.py` once to create the tables.
6. Run `execution.py` to start the application.
